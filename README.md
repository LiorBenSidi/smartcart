# SmartCart – Base44 App

**SmartCart** is an AI-powered grocery assistant that analyzes shopping receipts, tracks spending patterns, and provides personalized recommendations to help users save money and make healthier choices.

The system combines receipt analysis, behavioral modeling, and AI-based reasoning to turn everyday grocery data into clear, actionable insights.

## Creating a New User Account

To explore the system from a cold-start perspective:

1. Open the app:  
   https://smart-cart-technion.base44.app

2. Click **“Sign Up”** on the landing page.
   <img width="732" height="780" alt="image" src="https://github.com/user-attachments/assets/d0dfda2e-0f40-4151-ae7d-b880a3b32763" />


4. In the Base44 authentication modal, choose one of the following:
   - **Single Sign-On (SSO):** Google / Microsoft / Facebook  
   - **Email & Password:** Click **“Sign Up”** at the bottom of the modal
   <img width="1446" height="2144" alt="CleanShot 2026-01-27 at 13 46 38@2x" src="https://github.com/user-attachments/assets/ac61195b-e567-4b2b-9a48-09fa7d9063bb" />


5. If you choose email signup, you will receive a verification code by email, which must be entered to complete registration.

- After signing up as a new user, you will be guided through the **Onboarding Flow**.
- Afterwards, you will have full access to the app, allowing you to upload receipts and see how personalization improves as more data is added.
